# Weather Prediction Project
This project uses Machine Learning to predict weather conditions based on input data. It helps analyze patterns and forecast future weather.

# Features
* Predict temperature
* Analyze weather data
* Uses Machine Learning models

# Technologies Used
* Python 
* Machine Learning
* Libraries (NumPy, Pandas, Scikit-learn)
  
# Installation
1. Install Python
2. Install required libraries:
   pip install numpy pandas scikit-learn
   
# How to Run
python weather_prediction.py
